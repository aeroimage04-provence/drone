# AeroImage 04

Site vitrine public d’AeroImage 04, spécialisé dans les films et prises de vue par drone dans les Alpes-de-Haute-Provence.

## Publication avec GitHub Pages

1. Ouvrir **Settings** dans le dépôt.
2. Choisir **Pages** dans la colonne de gauche.
3. Sous **Build and deployment**, choisir **Deploy from a branch**.
4. Sélectionner la branche **main** et le dossier **/(root)**.
5. Cliquer sur **Save**.

Le site sera ensuite disponible à l’adresse `https://aeroimage04-provence.github.io/drone/`.
